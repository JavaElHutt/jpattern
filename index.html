Sewing pattern generator developed in Javascript!

JPattern is a small web application that allows you to develop the three basic sewing patterns. It was developed as a personal project, to learn how to use the HTML <canvas> element and thus be able to create graphics on a web page. The implemented javascript library can be easily extended, which provides the possibility of making more complex patterns. I hope this mini application helps you!.
